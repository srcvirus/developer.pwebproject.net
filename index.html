The pWeb Home Agent REST API must be accessed using HTTP GET requests. We utilize the HTTP status code in the reply message to indicate whether the API call was properly formatted. If the URI is properly formated a status code containing “200 OK” is returned. In case of a malformed GET request “400 Bad Request” is returned. Data is returned in JSON format. Details can be found in the documentation page.
